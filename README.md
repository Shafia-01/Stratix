## ✨ Features
KeyLytics provides a comprehensive suite of SEO intelligence tools powered by AI. Here's what each feature offers:

### 1. 🔍 Keyword Discovery
**AI-Powered Keyword Research with Real-Time Metrics**

Discover high-value keywords using advanced AI technology:
- **Multi-Source Intelligence**: Combines DataForSEO API (real-time data) with Gemini AI fallback for comprehensive coverage
- **Intelligent Metrics**: Get search volume, competition level, CPC (Cost Per Click), and trend scores for each keyword
- **Scalable Analysis**: Choose between Quick Analysis (5 keywords) or Comprehensive Analysis (50+ keywords)
- **Smart Scoring**: AI-powered keyword scoring system ranks keywords by opportunity potential
- **Batch Processing**: Parallel processing for fast results on large keyword sets
- **Database Integration**: Automatically saves discovered keywords to MySQL for future reference

**Use Cases**: Initial keyword research, expanding keyword lists, finding long-tail opportunities, identifying trending keywords

### 2. 📊 SERP Analysis
**Deep Dive into Search Engine Results Pages**

Analyze what's actually ranking and identify optimization opportunities:
- **Snippet Analysis**: Evaluates title tags, meta descriptions, and featured snippets for optimization opportunities
- **People Also Ask (PAA)**: Extracts and analyzes PAA questions to identify content gaps
- **Top Ranking Analysis**: Analyzes top-ranking pages for patterns, content structure, and ranking factors
- **Content Gap Identification**: Finds missing topics and content opportunities in current SERP results
- **Optimization Suggestions**: AI-generated recommendations for improving rankings
- **Competition Assessment**: Evaluates competition level and difficulty for ranking

**Use Cases**: Optimizing existing content, identifying featured snippet opportunities, understanding competitor strategies, finding content gaps

### 3. 🧩 Competitor Gap Analysis
**Find Keywords Your Competitors Rank For (But You Don't)**

Discover missed opportunities by analyzing competitor keyword strategies:
- **Competitor Identification**: Automatically identifies top competitors for your seed keyword
- **Keyword Gap Detection**: Finds keywords where competitors rank in top 20 but you don't
- **Gap Scoring**: Calculates opportunity scores based on competitor rankings and traffic potential
- **Traffic Potential Assessment**: Estimates traffic potential (high/medium/low) for each gap
- **Strategic Recommendations**: Prioritizes opportunities based on gap scores and feasibility
- **Ranking Analysis**: Shows exactly where competitors rank for each keyword

**Use Cases**: Competitive intelligence, finding easy wins, identifying market gaps, outranking competitors

### 4. 📝 Content Optimization
**AI-Powered Content Analysis and Optimization**

Optimize your existing content for better search performance:
- **Meta Title Suggestions**: Generates 3 optimized meta title options with SEO best practices
- **Meta Description Suggestions**: Creates compelling meta descriptions that improve click-through rates
- **Missing Topics Identification**: Identifies topics and subtopics you should cover for comprehensive content
- **Keyword Optimization Tips**: Provides specific recommendations for keyword placement and density
- **Content Structure Improvements**: Suggests improvements to headings, structure, and readability
- **Content Type Support**: Optimizes for Blog Posts, Product Pages, Landing Pages, Articles, and Guides

**Use Cases**: Improving existing content, optimizing new content before publishing, A/B testing meta tags, enhancing content comprehensiveness

### 5. 🎯 Search Intent Classification
**Understand What Users Really Want**

Classify keywords by search intent to align content with user needs:
- **Intent Types**: Classifies keywords into Informational, Commercial, Transactional, and Navigational intent
- **Hybrid Classification**: Combines rule-based detection with AI-powered refinement for accuracy
- **Intent Caching**: Stores intent classifications in database to avoid redundant analysis
- **Smart Fallbacks**: Uses Gemini AI when rule-based classification is uncertain
- **Intent-Based Insights**: Provides recommendations based on intent type for better content alignment

**Use Cases**: Content strategy planning, aligning content with user intent, improving conversion rates, optimizing for different funnel stages

### 6. 💰 Conversion Mapping
**Rank Keywords by ROI Potential**

Prioritize keywords that drive the highest return on investment:
- **ROI Calculation**: Calculates ROI potential using formula: Score / (CPC + 0.01)
- **Buyer Intent Ranking**: Ranks keywords by conversion potential and commercial value
- **CPC Analysis**: Identifies high-value keywords with optimal cost-per-click ratios
- **Visual ROI Charts**: Interactive charts showing top keywords by ROI potential
- **Conversion-Focused Filtering**: Filters and sorts keywords for maximum conversion value
- **Quick Analysis Mode**: Rapid conversion analysis for specific keywords

**Use Cases**: PPC campaign planning, prioritizing high-converting keywords, budget allocation, identifying high-ROI opportunities

### 7. 🧠 Topic Clustering
**Group Keywords into Semantic Topic Clusters**

Organize keywords into meaningful topic groups for content planning:
- **Semantic Clustering**: Uses AI to group keywords by semantic similarity and topic relevance
- **Cluster Enhancement**: Enhances clusters with metrics (volume, competition, CPC) for each keyword
- **Topic Insights**: Generates insights and recommendations for each cluster
- **Content Strategy Mapping**: Maps clusters to content strategy and pillar pages
- **Cluster Scoring**: Ranks clusters by opportunity potential and keyword value
- **Visual Organization**: Organizes keywords into logical topic groups for easy content planning

**Use Cases**: Content pillar planning, topic research, content calendar creation, semantic SEO strategy, organizing keyword research

### 8. 🌐 Industry Focus
**Industry-Specific Keyword Intelligence**

Get tailored keyword recommendations for your specific industry:
- **Industry Selection**: Choose from 15+ industries (Technology, Healthcare, Finance, E-commerce, etc.)
- **Industry-Specific Keywords**: Generates 20+ high-value keywords tailored to your industry
- **Trend Awareness**: Includes current trends and opportunities specific to your industry
- **Intent Mix**: Provides mix of informational, commercial, and transactional keywords
- **Industry Insights**: Generates industry-specific insights and opportunities
- **Customizable**: Adapts keyword recommendations based on industry characteristics

**Use Cases**: Industry research, niche keyword discovery, industry trend analysis, competitive industry analysis

### 9. 📈 Trend Forecasting
**Predict Keyword Trends with Seasonal Analysis**

Forecast keyword performance and identify seasonal patterns:
- **6-Month Forecasts**: Predicts keyword trends for the next 6 months with confidence scores
- **Seasonal Pattern Analysis**: Identifies seasonal trends and optimal content timing
- **Growth Rate Calculations**: Calculates trend direction and growth rates
- **Market Opportunity Identification**: Identifies trending keywords before they peak
- **Historical Trend Data**: Analyzes historical performance to predict future trends
- **Trend Insights**: Provides actionable insights on when to create content for maximum impact

**Use Cases**: Content calendar planning, seasonal content strategy, identifying trending topics, timing content publication

### 10. 🧩 Full Strategy
**Comprehensive SEO Strategy in One Analysis**

Run all modules together for a complete SEO strategy:
- **Unified Analysis**: Combines Keyword Discovery, Competitor Gap, Topic Clustering, Trend Forecasting, and SERP Analysis
- **Executive Summary**: Provides high-level overview of all analyses
- **Integrated Dashboard**: Single dashboard with all analysis results in one place
- **Strategic Recommendations**: Comprehensive recommendations based on all analyses
- **Progress Tracking**: Real-time progress updates as each module runs
- **Complete Picture**: Get the full SEO strategy picture for your keyword in one analysis

**Use Cases**: Comprehensive SEO audits, new market entry, complete keyword strategy, strategic planning, client reporting

## 🔧 Configuration
### Environment Variables
Create a `.env` file in the project root with your API keys and database configuration:

```bash
# Required API Keys
GEMINI_API_KEY=your_gemini_api_key
SERPAPI_KEY=your_serpapi_key

# Optional - Enhanced Keyword Research (DataForSEO)
# DataForSEO works in the background and automatically falls back to Gemini if unavailable
DATAFORSEO_USERNAME=your_dataforseo_username
DATAFORSEO_PASSWORD=your_dataforseo_password

# DataForSEO Credit Preservation Settings (optional)
DATAFORSEO_DEMO_MODE=false             # Set to true to use sandbox only (preserves credits)
DATAFORSEO_FORCE_SANDBOX=false         # Set to true to force sandbox mode
DATAFORSEO_PRESERVE_CREDITS=true       # Auto-switch to sandbox when balance is low
DATAFORSEO_LOW_BALANCE_THRESHOLD=0.50  # Switch to sandbox when balance < $0.50

# Optional - MySQL Database Configuration
MYSQL_HOST=127.0.0.1
MYSQL_USER=root
MYSQL_PASSWORD=your_password
MYSQL_DATABASE=gemkey_ai
MYSQL_PORT=3306
```

### System Status Monitoring
The application displays real-time API status in the sidebar:
- **Gemini API**: Status shown in System Status panel
- **SerpAPI**: Status shown in System Status panel
- **DataForSEO**: Works silently in the background
  - Automatically falls back to Gemini if DataForSEO is unavailable
  - Uses sandbox mode when balance is low or in demo mode

### DataForSEO Integration (Optional)
DataForSEO provides enhanced keyword research with **real-time, live data**. It works automatically in the background:

1. **Setup**: Create a DataForSEO account at https://dataforseo.com/
2. **Configuration**: Add your credentials to `.env`
3. **Automatic Management**: The system intelligently manages API usage:
   - **Primary**: Uses Live DataForSEO API for real-time keyword data
   - **Auto-Switch**: Automatically switches to Sandbox API if account is limited or balance = 0
   - **Fallback**: Falls back to Gemini AI if DataForSEO is completely unreachable
4. **Error Handling**: The system automatically detects and handles:
   - Account balance issues (HTTP 402)
   - Rate limiting (HTTP 429)
   - Network connectivity issues
   - Authentication errors
   - API errors

**Note**: DataForSEO status is not displayed in the System Status panel. It operates silently in the background and automatically uses fallbacks when needed.

### 💰 Credit Preservation (For Testing/Demo)
To preserve your DataForSEO credits for presentations while testing:

**Option 1: Demo Mode (Recommended for Testing)**
```bash
# In your .env file:
DATAFORSEO_DEMO_MODE=true
```
This uses sandbox/mock data - perfect for testing without consuming credits.

**Option 2: Force Sandbox Mode**
```bash
# In your .env file:
DATAFORSEO_FORCE_SANDBOX=true
```
Forces sandbox mode regardless of balance - preserves all credits.

**Option 3: Auto-Preserve (Default)**
```bash
# In your .env file:
DATAFORSEO_PRESERVE_CREDITS=true
DATAFORSEO_LOW_BALANCE_THRESHOLD=0.50
```
Automatically switches to sandbox when balance drops below threshold.

**For Presentation:**
Simply set `DATAFORSEO_DEMO_MODE=false` and `DATAFORSEO_FORCE_SANDBOX=false` to use live API with your remaining credits.

## 🚨 Troubleshooting
### Common Issues
1. **API key errors**
   - Verify API keys in `.env` file
   - Check API quotas and limits
   - View System Status in the sidebar to see API connection status
   - Test API endpoints manually

2. **API Status shows Offline**
   - Check your internet connection
   - Verify API keys are correct in `.env` file
   - Check API service status (Google AI Studio, SerpAPI status page)
   - Click "Retest APIs" button if available
   - Restart the Streamlit app

3. **Streamlit not loading**
   ```bash
   streamlit run app.py --server.port 8502
   ```

4. **Module import errors**
   ```bash
   pip install -r requirements.txt
   ```

5. **Database connection errors**
   - Verify MySQL is running
   - Check database credentials in `.env` file
   - Ensure the database `gemkey_ai` exists
   - Create database: `CREATE DATABASE gemkey_ai;`

6. **DataForSEO not working**
   - Verify credentials are correct in `.env` file
   - Check account balance on DataForSEO dashboard
   - System will automatically fallback to Gemini if DataForSEO is unavailable
   - Enable sandbox mode for testing: `DATAFORSEO_DEMO_MODE=true`
